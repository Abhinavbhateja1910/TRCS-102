# Day 03 - Introduction to CSS and Styling Employee Registration Form

**Date:** 24 June 2026

## Overview

Today was the third day of my Web Development Training. The focus of today's session was on CSS (Cascading Style Sheets). I learned how CSS is used to style web pages and improve their appearance. CSS helps separate the design from the structure of a webpage, making websites more attractive and user-friendly.

To apply the concepts learned today, I styled the Employee Registration Form that I created previously using CSS.
