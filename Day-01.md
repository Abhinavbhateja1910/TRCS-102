# Day 01 - Introduction to HTML

**Date:** 22 June 2026

## Overview

Today was my first day of Web Development Training. I was introduced to the basics of web development and learned about HTML, which is the foundation of every webpage. The session focused on understanding how web pages are structured and how HTML elements are used to display content in a browser.

## Topics Learned

- Introduction to Web Development
- What is HTML (HyperText Markup Language)
- Structure of an HTML Document
- Basic HTML Tags
- Headings (`<h1>` to `<h6>`)
- Paragraph Tag (`<p>`)
- Line Break Tag (`<br>`)
- Horizontal Rule Tag (`<hr>`)
- Text Formatting Tags
- Ordered and Unordered Lists
- Comments in HTML

## Practical Work

During the practical session, I:

- Created my first HTML webpage.
- Learned how to use the `<!DOCTYPE html>` declaration.
- Created headings and paragraphs.
- Practiced text formatting tags such as bold, italic, and underline.
- Created ordered and unordered lists.
- Added comments to improve code readability.
- Viewed the webpage in a browser and observed the output.

## Key Learnings

- HTML is the standard markup language used to create web pages.
- Every HTML document follows a specific structure consisting of `<html>`, `<head>`, and `<body>` tags.
- Headings help organize content hierarchically.
- Lists are useful for presenting information in a structured format.
- Comments help developers document their code without affecting the webpage output.

## Challenges Faced

- Remembering the syntax of different HTML tags.
- Understanding the purpose of the `<head>` and `<body>` sections.

## How I Overcame Them

- Practiced writing HTML code multiple times.
- Reviewed examples and compared the browser output with the source code.

## Conclusion

The first day provided a strong foundation for web development. I gained a basic understanding of HTML and learned how webpages are created and structured. The hands-on practice helped me become familiar with writing and running simple HTML programs.
