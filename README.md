# TRCS-102 (4 Week Web Development Training)
## Introduction

Welcome to my Web Development Training Diary!

This repository serves as a day-by-day record of my learning journey during my web development training. The purpose of this repository is to document the concepts I learn, the projects I build, the challenges I encounter, and the solutions I discover throughout the training period.

By maintaining this diary, I aim to:
- Track my daily progress and learning milestones.
- Build a habit of documenting technical work.
- Create a reference for future revision.
- Showcase my consistency and practical experience in web development.

The repository contains separate Markdown files for each training day, detailing the topics covered, tasks completed, key learnings, and future goals.
