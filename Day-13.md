## Key Learnings

- A responsive dashboard improves usability on all devices.
- CSS media queries help adapt layouts for different screen sizes.
- Proper spacing and alignment enhance the overall user experience.
- Responsive design should be implemented consistently across all pages of a web application.
- Testing on multiple devices helps identify layout issues early.

---

## Challenges Faced

- Rearranging dashboard elements for smaller screens.
- Maintaining readability without affecting the desktop layout.
- Ensuring consistent spacing and alignment across different devices.

---

## How I Overcame Them

- Used CSS media queries to define breakpoints.
- Modified Flexbox properties for smaller screen sizes.
- Adjusted margins, padding, and widths to improve responsiveness.
- Tested the dashboard repeatedly using browser developer tools and refined the layout until it displayed correctly on all devices.
