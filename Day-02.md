# Day 02 - HTML Forms and Tables

**Date:** 23 June 2026

## Overview

Today was the second day of my Web Development Training. The focus of today's session was on HTML Forms and Tables. I learned how forms are used to collect user input and how tables help in displaying data in a structured format. These concepts are essential for developing web applications and management systems.

As part of my learning, I also explored how forms and tables can be used in an Employee Management System to collect and display employee information.

## Topics Learned

- HTML Forms
- Form Elements
  - `<form>`
  - `<input>`
  - `<label>`
  - `<textarea>`
  - `<select>`
  - `<option>`
  - `<button>`
- Input Types
  - Text
  - Email
  - Number
  - Password
  - Radio Button
  - Checkbox
  - Date
  - File
- Form Attributes
  - action
  - method
  - placeholder
  - required
- HTML Tables
  - `<table>`
  - `<tr>`
  - `<th>`
  - `<td>`
  - `<caption>`
  - `<thead>`
  - `<tbody>`
- Table Attributes
  - rowspan
  - colspan

- Tables provide an organized way to display information.
- Forms and tables are fundamental components of management systems and database-driven applications.
