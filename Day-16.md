# Day 16 - Employee List Page and Database Data Retrieval

**Date:** 13 July 2026

## Overview

Today was the sixteenth day of my Web Development Training. The focus of today's session was on creating the **Employee List** page for the Employee Management System.

I developed a new page named **employee.php**, which displays the details of all registered employees in a tabular format. To achieve this, I connected the page to the MySQL database using PHP and retrieved all employee records using the SQL `SELECT` statement. This allowed the application to dynamically display employee information stored in the database.

This was an important milestone because the Employee Management System can now display all employee records in a structured and organized manner.

## Employee Management System Progress

Today I added a new module to my Employee Management System for viewing employee records.

### Features Implemented

- Employee List Page (`employee.php`)
- PHP and MySQL Database Connection
- Dynamic Employee Records Table
- Display of All Registered Employees
- Automatic Data Retrieval from Database
- Structured and User-Friendly Table Layout

### Employee Details Displayed

- Employee ID
- Employee Name
- Email Address
- Mobile Number
- Salary
- Date of Registration

### Workflow

1. User opens the **employee.php** page.
2. PHP establishes a connection with the MySQL database.
3. A SQL `SELECT` query retrieves all employee records.
4. PHP fetches each row from the database.
5. The records are displayed dynamically in an HTML table.
6. Any newly registered employee is automatically shown in the list.
